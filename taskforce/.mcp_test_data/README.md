# MCP Test Data Directory

This directory contains sample files for validating the MCP filesystem server integration.

## Files

- `sample.txt` - Plain text file for read testing
- `data.json` - JSON file for structured data testing
- `README.md` - This file (markdown example)

## Purpose

Used by Story 2.3 to validate that the Taskforce agent can successfully:
1. List files in this directory via MCP tools
2. Read file contents via MCP tools
3. Integrate MCP tools alongside native tools
